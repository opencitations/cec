CITATION EXTRACTOR – CEX

This tool allows you to upload a PDF file, process it using Grobid and obtain a JSON file containing all the sentences where an intext citation is present. Both the grobid output (XML-TEI) and the JSON can be downloaded.

By clicking on “Process File”, Grobid will perform the processing of the input PDF and as a result a zip folder, containing the Grobid xml and a JSON file composed by the citations sentences, will be automatically downloaded to your machine. Be patient, it will take few seconds.

N.B. You can process only ONE PDF at the time.

You can choose to perform the semantic alignment of the original article's section headings by ticking the box above. When this option is selected, the JSON file, which contains all the sentences with in-text citations, is enriched with a new key called "ALIGNED SECTION." This key stores the aligned section title if it is found.
