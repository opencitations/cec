PDF Processor

Overview

This tool allows you to upload a PDF file, process it using Grobid
and obtain a JSON file containing all the sentences
where an intext citation is present. Both the grobid output (XML-TEI)
and the JSON can be downloaded. N.B. You can process only ONE PDF at the time.
